SH/Renepo-A 'Opener' malware targets Mac OS X (MacCentral) MacCentral - As first reported by Macintouch last week, there's a new form of malware that specifically targets Mac OS X. Already dubbed "Opener," it's been confirmed by corporate anti-virus software maker Sophos Plc., which has posted details on its Web site, and has called the malware "SH/Renepo-A." Sophos describes the code as a shell script worm that specifically targets Mac OS X computers.
